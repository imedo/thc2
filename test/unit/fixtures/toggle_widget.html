Different kinds of toggle widgets

  • Link and target determined by DOM structure

    Toggle me (automatic target)

    This is a toggle box where toggle link and toggle target are discovered automatically using the DOM structure.

  • Target specified by class parameter

    Toggle me (explicit target)

    This is a toggle box where toggle link and toggle target are connected explicitly.

  • Link and Target specified in Javascript code

    Toggle me (specified in JS)

    This is a toggle box where toggle link and toggle target are connected in Javascript.
